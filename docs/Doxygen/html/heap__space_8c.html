Detailed Description

This file implements the heap-space interface This implementation adds HEAP instrumentation to the internal allocators.

Copyright
Please read the Exosite Copyright: copyright

This file also implements the heap_space_internal interface The implementation routes the allocator calls to

  • default allocator call
  • registered allocator call
  • internal memspace

If there was no allocator registered and heap_space is NULL then the calls are routed to the default allocator. If there was an allocator registered and heap_space is NULL then the calls are routed to the registered allocator. If heap_space is not NULL then the calls are routed to mspace allocator
